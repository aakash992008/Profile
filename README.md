## Enable GitHub Pages

1. Go to your repo: https://github.com/aakash992008/Profile
2. Click **Settings** → **Pages**
3. Under "Source", select **Deploy from a branch**
4. Branch: **main** (or **master**), folder: **/ (root)**
5. Click **Save**

Your site will be live at `https://aakash992008.github.io/Profile/` within a few minutes.
